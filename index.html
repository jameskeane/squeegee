Introducing Squeegee.js

Squeegee.js is a simple javascript library that can be embedded directly into your SVG documents to provide panning and deep zoom functionality. Squeegee does for SVG what Zoom.it does for static images.

Usage: Simply include the squeegee javascript in your SVG file, and give the desired object an id named 'viewport'

...
<script xlink:href="svgpan-min.js"/>
<g id="viewport" ...

Example

Give it a try, use your mouse to pan the image and scroll wheel to zoom. Just like google maps, except there is no limit to the zoom depth!

No tricks involved, check out the raw SVG.
